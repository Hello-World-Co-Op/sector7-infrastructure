## Step 1: Download AppFlowy Desktop

- [ ] Go to: https://appflowy.io/download
- [ ] Download the version for your operating system:
  - **Windows**: `.exe` installer
  - **Mac**: `.dmg` file
  - **Linux**: `.AppImage` or `.deb`
- [ ] Install the application

---

## Step 2: Configure Self-Hosted Server

When you first open AppFlowy, you need to point it to our self-hosted server:

- [ ] Open AppFlowy
- [ ] Click **"Quick Start"** or look for settings
- [ ] Go to **Settings** (gear icon) → **Cloud Settings**
- [ ] Select **"AppFlowy Cloud Self-hosted"**
- [ ] Enter the server URL:
  ```
  https://appflowy.sector7.helloworlddao.com
  ```
- [ ] Click **Save** or **Connect**

---

## Step 3: Create Your Account

- [ ] Click **"Sign Up"** (not Sign In)
- [ ] Enter your email address

- [ ] Complete the registration

**Important**: Your account will be auto-confirmed (no email verification needed).

## Troubleshooting

### "Connection failed" error
- Check that you're connected to the internet
- Verify the URL is exactly: `https://appflowy.sector7.helloworlddao.com`
- Try restarting the app

### "Invalid credentials" error
- Make sure you're using Sign Up (not Sign In) for first-time setup
- Password requirements: minimum 8 characters, mix of upper/lower case, numbers

### Can't see shared workspace
- Contact Graydon to confirm you've been added
- Try logging out and back in
- Check Settings → Cloud Settings to verify server connection
